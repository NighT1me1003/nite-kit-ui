# Contributing

Thank you for your interest in NiteKit UI.

## Review Policy

This is a public repository, but direct push access is limited to maintainers or explicitly authorized collaborators.

External changes should be submitted through Pull Requests or Merge Requests. Changes are reviewed before they are merged.

## Local Setup

```bash
npm install
npm run dev
```

## Checks

Before submitting a change, run:

```bash
npm run typecheck
npm run build:lib
```

## Scope

Good contributions include:

- Bug fixes for existing components.
- Accessibility improvements.
- Documentation improvements.
- Focused component examples.
- Small visual fixes that match the current design language.

Please avoid unrelated refactors in the same change.
