# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-21

### Added

- Initial public release of `nitekit`.
- Vue 3 plugin installation through `app.use(NiteKit)`.
- TypeScript declaration output.
- Core components: `NkButton`, `NkCard`, `NkInput`, `NkPagination`, `NkProgress`, `NkSwitch`, `NkTable`, and `NkTag`.
- Extended components including form, navigation, feedback, data display, and layout primitives.
- Bilingual README files and documentation portal.
